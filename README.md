# Mocha + Chai BDD Pack for ASP.NET MVC

Provides the initial skeleton to write specs/tests for client side scripts in ASP.NET MVC Application.

It includes the following files in an ASP.NET MVC Application:

    |-YourProject
       |-Content
       |---mocha.css
       |-Controllers
       |---MochaController.cs
       |-Scripts
       |---specs
       |-----bdd
       |-------chai.js
       |-------mocha.js
       |-----calculator.js
       |-Views
       |---Mocha
       |-----Index.cshtml
